# Eva Public Demo

Eva's Phase 29 public demo is a deterministic local report/status profile.

It presents the command system, bounded planner and agent loop, capability registry, locked execution gates, and completed preview foundations without enabling new execution.

The demo does not publish, upload, install, commit, tag, push, call live providers, control a browser or desktop, edit source, run shell commands, or expose secrets.

Start with `eva release status`, continue with `eva release demo`, and use the capability, safety, limitation, and verification reports as evidence.

Browser behavior is public-URL read-only observation only. Desktop behavior is observation-only, with control limited to dry-run policy reports.

News remains local/mock or safe-read-only. CodingAgent remains preview/report/status only. Voice remains a locked/mock foundation.

Phase 12L narrow approved new `.md`/`.txt` creation remains the only real file write path.

The next safe step is Release Candidate Hardening / optional user-approved commit planning.
